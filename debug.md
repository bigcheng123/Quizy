# Quizy 冒烟测试规则

> 每次修改项目代码后，按本文档流程逐项验证。未通过的项目请标记并修复后再提交。

---

## 一、环境准备

### 1.1 清理运行数据（可选，按需执行）

仅当修改涉及数据库结构、种子文件或配置 schema 时需要重置：

```
删除 %APPDATA%/Quizy/quizy.db（及同目录 .db-shm / .db-wal）
删除 %APPDATA%/Quizy/config.json
```

### 1.2 安装依赖

```bash
npm install
```

**验证点**：安装无报错，`postinstall` 中 `electron-rebuild` 成功编译 `better-sqlite3`。

### 1.3 运行自动化测试

```bash
npm test
```

**验证点**：全部 9 个测试用例 PASS（`db.test.js` 8 个 + `store.test.js` 2 个），无 FAIL。

---

## 二、开发模式启动

```bash
npm run dev
```

**验证点**：
- [ ] Quiz 窗口全屏启动，无边框，覆盖整个屏幕
- [ ] Windows 任务栏不可见或被 Quiz 窗口遮挡
- [ ] DevTools 以独立窗口打开（detached 模式）
- [ ] 控制台无红色报错（仅允许关于 audio autoplay policy 的黄色警告）

## 三、答题窗口（Quiz Window）冒烟测试

### 3.1 界面渲染

- [ ] **星空背景**：深蓝渐变背景可见，约 80 个星点在闪烁
- [ ] **进度条**：顶部显示"语文"和"数学"两行，初始各显示 `☆☆☆☆☆` 和 `0/5`
- [ ] **科目切换标签**："语文"/"数学"两个按钮，当前选中高亮
- [ ] **题目卡片**：白色半透明圆角卡片，包含年级标签、题型标签、题目内容
- [ ] **选项按钮**：根据题型显示 A/B/C/D 按钮 或 "✓ 正确"/"✗ 错误"按钮 或 填空输入框
- [ ] **设置按钮**：右下角可见 "⚙️ 设置" 按钮
- [ ] **右键菜单**：在页面上右键无反应（已被屏蔽）

### 3.2 答题流程 — 答对

1. 阅读题目，选择正确答案
2. **验证点**：
   - [ ] 选项按钮变绿（`.correct` 样式）
   - [ ] 屏幕中央浮现 ⭐ emoji 并缩放入场 + 淡出（约 0.8 秒）
   - [ ] 听到答对音效（短促清脆铃声）
   - [ ] 进度条对应科目星星从 ☆ 变为 ⭐，数字递增（如 `1/5`）
   - [ ] 约 0.9 秒后自动加载下一题

### 3.3 答题流程 — 答错

1. 阅读题目，故意选择错误答案
2. **验证点**：
   - [ ] 选项按钮变红（`.wrong` 样式）
   - [ ] 正确答案被高亮为绿色
   - [ ] 题目卡片抖动（`shake` 动画，约 0.4 秒）
   - [ ] 屏幕中央浮现 💔 emoji（0.8 秒缩放淡出）
   - [ ] 听到答错音效
   - [ ] 进度条不变（答错不增加星星计数）
   - [ ] 约 1.2 秒后自动加载下一题

### 3.4 答题中切换科目

1. 在语文题目显示时，点击"数学"标签
2. **验证点**：
   - [ ] 如果正在答题等待中（isAnswering=true），标签点击无反应
   - [ ] 如果非等待中，正常切换到数学题目
   - [ ] 标签高亮跟随切换

### 3.5 填空题交互

1. 当遇到填空题时
2. **验证点**：
   - [ ] 显示文本输入框和"确认"按钮
   - [ ] 输入框自动获得焦点
   - [ ] 点击"确认"或按 Enter 键提交答案
   - [ ] 空内容点击确认无反应
   - [ ] 答错后，输入框回填正确答案（绿色文字显示）

### 3.6 判断题交互

1. 当遇到判断题时
2. **验证点**：
   - [ ] 显示两个按钮："✓ 正确" 和 "✗ 错误"
   - [ ] 无 A/B/C/D 选项按钮

### 3.7 自动切换科目

1. 将语文答满所需题数（如 5 题全对）
2. **验证点**：
   - [ ] 答完最后一题语文后，自动切换到数学科目
   - [ ] 标签自动更新为"数学"高亮

### 3.8 解锁流程

1. 将语文和数学都答满所需题数
2. **验证点**：
   - [ ] 出现全屏解锁遮罩（`#unlock-overlay`）
   - [ ] 显示 🎉 emoji、"全部答对，通过！"标题和鼓励文案
   - [ ] 显示"结束锁屏，进入桌面"按钮
   - [ ] 点击按钮后，Quiz 窗口关闭，应用退出（dev 模式下窗口直接消失）

### 3.9 解锁按钮防重复点击

1. 在解锁界面快速连续点击"结束锁屏，进入桌面"按钮
2. **验证点**：
   - [ ] 第二次点击无反应（`unlockExitRequested` 防护）
   - [ ] 按钮文案变为"正在退出…"且按钮禁用

## 四、管理后台（Admin Window）冒烟测试

### 4.1 管理员入口

1. 点击右下角"⚙️ 设置"按钮
2. **验证点**：
   - [ ] 弹出半透明黑色遮罩 + 居中白色弹窗
   - [ ] 弹窗标题"管理员验证"
   - [ ] 密码输入框自动获得焦点
   - [ ] 输入框为 `type="password"`（输入内容不可见）

### 4.2 密码验证 — 正确

1. 输入默认密码 `123456`，点击"确认"或按 Enter
2. **验证点**：
   - [ ] 弹窗消失
   - [ ] 管理后台窗口打开（900×700，带标题栏，可调整大小）
   - [ ] 后台窗口也在最前（`alwaysOnTop`）
   - [ ] Quiz 窗口没有抢回焦点（blur handler 识别到 adminWindow 存在后跳过 refocus）

### 4.3 密码验证 — 错误

1. 关闭后台窗口，重新点击"⚙️ 设置"，输入错误密码
2. **验证点**：
   - [ ] 显示红色文字"密码错误，请重试"
   - [ ] 密码输入框被清空并获得焦点
   - [ ] 按 Escape 键可关闭弹窗

### 4.4 管理后台 — Tab 切换

1. 在后台窗口点击三个导航按钮："基础设置"、"题库管理"、"答题记录"
2. **验证点**：
   - [ ] 点击标签时，对应 tab-content 显示，其余隐藏
   - [ ] 导航按钮 `.active` 样式跟随切换
   - [ ] 切换到"题库管理"时自动加载题目列表
   - [ ] 切换到"答题记录"时自动加载日期下拉

### 4.5 基础设置 — 读取并修改配置

1. 检查初始值
2. **验证点**：
   - [ ] 年级下拉默认"三年级"
   - [ ] 科目多选：语文、数学均选中
   - [ ] 题型多选：选择、判断、填空、看图均选中
   - [ ] "答对的题目不再出现"复选框选中
   - [ ] 语文题量 = 5，数学题量 = 5
   - [ ] 密码字段为空

3. 修改：年级改为"五年级"，语文题量改为 3，数学题量改为 3，点击"保存设置"
4. **验证点**：
   - [ ] 显示绿色"保存成功。"提示，3 秒后消失
   - [ ] 题库统计自动刷新（显示五年级对应科目题数）
   - [ ] 关闭后台，Quiz 答题页的进度条语文和数学均变为 3 颗星（`0/3`）

### 4.6 基础设置 — 修改密码

1. 新密码和确认密码均输入 `666666`，点击"保存设置"
2. **验证点**：
   - [ ] 保存成功
   - [ ] 下次进入后台需输入新密码 `666666`（旧密码 `123456` 失效）

### 4.7 基础设置 — 输入校验

1. 语文题量输入 0 或 21，点击保存
2. **验证点**：显示红色错误提示"语文每日题量须为 1–20 的整数。"
3. 新密码输入 `111`，确认密码输入 `222`，点击保存
4. **验证点**：显示红色错误提示"两次输入的新密码不一致。"

### 4.8 基础设置 — 科目联动

1. 取消勾选"数学"科目，保存
2. **验证点**：
   - [ ] 数学题量输入框隐藏
   - [ ] 关闭后台后，Quiz 页面数学标签和进度条隐藏，只显示语文
   - [ ] 只需答完语文即可解锁

### 4.9 基础设置 — 题库统计

- [ ] `.stats-grid` 显示当前选中科目的统计卡片（每科×当前年级 一张，如全选则是 2 张）
- [ ] 卡片显示题数和"X年级 · 语文/数学"标签

### 4.10 基础设置 — 强制结束程序

1. 点击 header 右侧"强制结束程序"按钮
2. **验证点**：
   - [ ] 弹出确认对话框
   - [ ] 确认后 Quizy 完全退出（quiz 窗口 + admin 窗口均关闭）

## 五、题库管理（Questions Tab）冒烟测试

### 5.1 题目列表

- [ ] 切换到"题库管理"标签后，题目列表自动加载
- [ ] 每条显示：编号、科目标签、年级标签、题型标签、题干（最多 50 字截断）、答案（最多 20 字截断）、编辑/删除按钮
- [ ] 每条显示答对率统计（答对率百分比 + 答对次数/总次数）

### 5.2 筛选

1. 科目选"语文"→ 点击"筛选"
2. **验证点**：列表仅显示语文题目
3. 年级选"一年级"→ 点击"筛选"
4. **验证点**：列表仅显示一年级语文题目
5. 类型选"选择题"→ 点击"筛选"
6. **验证点**：列表仅显示一年级语文选择题

### 5.3 新增题目 — 选择题

1. 点击"+ 新增题目"
2. **验证点**：表单展开，标题"新增题目"，4 个选项输入框可见
3. 填写：科目=语文、年级=二年级、类型=选择题、题干="测试选择题"、4 个选项均填写、答案写一个选项的内容
4. 点击"保存"
5. **验证点**：
   - [ ] 显示"保存成功。"，表单收起
   - [ ] 列表刷新，出现新题
   - [ ] 切换到"基础设置"，题库统计中二年级语文题数 +1
   - [ ] 回到 Quiz 答题页，切换到语文，可能抽到这道新题

### 5.4 新增题目 — 判断题

1. 点击"+ 新增题目"，类型选"判断题"
2. **验证点**：选项输入框隐藏
3. 答案填"正确"或"错误"以外的内容，点击保存
4. **验证点**：显示"判断题答案请填「正确」或「错误」。"
5. 改为"正确"，保存成功

### 5.5 新增题目 — 填空题

1. 类型选"填空题"
2. **验证点**：选项输入框隐藏
3. 只填题干不填答案，点击保存
4. **验证点**：显示"请填写正确答案。"
5. 补填答案后保存成功

### 5.6 新增题目 — 选项校验

1. 类型选"选择题"，4 个选项中留一个为空，点击保存
2. **验证点**：显示"请填写完整的四个选项。"
3. 4 个选项均填，但答案与任一选项都不完全一致，点击保存
4. **验证点**：显示"正确答案必须与某一选项文案完全一致。"

### 5.7 编辑题目

1. 点击某道题的"编辑"按钮
2. **验证点**：表单展开，标题"编辑题目"，各字段回填原值
3. 修改题干内容，保存
4. **验证点**：列表刷新，该题题干显示新内容

### 5.8 删除题目

1. 点击某道题的"删除"按钮
2. **验证点**：弹出 `confirm` 确认对话框
3. 点击"取消"→ 无变化
4. 再次点击"删除"→ 点击"确定"→ 题目从列表消失，统计更新

### 5.9 取消表单

1. 点击"+ 新增题目"，填写部分内容
2. 点击"取消"按钮
3. **验证点**：表单收起，列表不变
