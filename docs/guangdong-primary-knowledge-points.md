## 一、语文（部编版）

### 一年级 ★

| 模块 | 知识点 |
|------|--------|
| 拼音 | 声母、韵母、整体认读；声调与标调；拼读 |
| 识字写字 | 基本笔画笔顺；偏旁部首；800—1000 字累计 |
| 阅读 | 朗读停顿；图文对应；联系上下文理解词义 |
| 写话 | 看图写一两句话 |
| 积累 | 《咏鹅》《江南》等；简单礼貌用语 |

### 二年级 ★

| 模块 | 知识点 |
|------|--------|
| 识字写字 | 1000—1600 字；多音字、形近字；部首查字法 |
| 阅读 | 默读；找关键词句；简单寓言、童话理解 |
| 写话/写段 | 日记、留言条；「谁干什么」句段 |
| 积累 | 《登鹳雀楼》《望庐山瀑布》等；谚语、对联入门 |

### 三年级 ★

| 模块 | 知识点 |
|------|--------|
| 识字写字 | 2500 字左右；钢笔书写 |
| 阅读 | 概括段意与主要内容；中心句；记叙文、说明文初识 |
| 习作 | 写人写事片段；比喻、拟人 |
| 积累 | 《山行》《赠刘景文》等；成语故事 |

### 四年级 ★

| 模块 | 知识点 |
|------|--------|
| 阅读 | 文章线索；体会情感；科普说明文 |
| 习作 | 书信、读后感；场面描写 |
| 口语 | 复述、讨论 |
| 积累 | 《出塞》《夏日绝句》等；名人名言 |

### 五年级 ★

| 模块 | 知识点 |
|------|--------|
| 阅读 | 详略安排；说明方法（举例子、列数字、打比方） |
| 习作 | 记叙文完整结构；想象作文、读后感 |
| 文言文 | 《自相矛盾》《杨氏之子》等；实词、翻译 |
| 积累 | 《示儿》《题临安邸》等；四大名著常识 |

### 六年级 ★

| 模块 | 知识点 |
|------|--------|
| 阅读 | 非连续性文本；综合理解；作者观点 |
| 习作 | 演讲稿、议论文片段；考场作文结构 |
| 文言文 | 《学弈》《两小儿辩日》《文言文二则》 |
| 字词句 | 关联词；病句修改；引号、破折号、省略号 |
| 总复习 | 1—6 年级古诗、名著、修辞综合 |

## 二、数学（人教版）

| 年级 | 数与运算 ★ | 图形与测量 ★ | 统计与应用 |
|------|------------|--------------|------------|
| **1** | 20 以内认识与加减；钟表整时半时 | 立体图形；位置上下前后左右 | 分类统计 |
| **2** | 100 以内加减；表内乘除；混合运算 | 厘米、米；直角；轴对称 | 简单数据收集 |
| **3** | 万以内加减；乘一位、除一位；分数小数初步 | 周长、面积；克千克 | 条形图；平均数初步 |
| **4** | 大数；三位数乘除；运算律；小数加减 | 角度量；三角形、平行四边形；垂直平行 | 行程问题；优化 |
| **5** | 因数倍数；分数四则；小数乘除；简易方程 | 体积表面积；旋转；组合图形面积 | 折线图；可能性 |
| **6** | 比与比例；百分数；负数 | 圆；圆柱圆锥；扇形统计图 | 比例尺；鸽巢问题；总复习 |

## 三、英语（人教版 PEP，多从三年级起）

| 年级 | 话题 ★ | 语法/句型 ★ |
|------|--------|-------------|
| **3** | 问候、数字、颜色、文具、动物 | What's this? It's…；How are you? |
| **4** | 教室、食物、家庭、天气 | 一般现在时；like/have |
| **5** | 星期、季节、位置、活动 | There is/are；现在进行时 |
| **6** | 交通、爱好、职业、计划 | 一般过去时；比较级 |

---

## 四、道德与法治（统编）

| 学段 | 要点 |
|------|------|
| **1—2 年级** | 校园规则、安全、家庭尊重、节日与传统文化 |
| **3—6 年级** | 宪法启蒙、权利义务、网络文明、生态文明、民族团结 |

---

## 五、科学（3—6 年级）

| 领域 | 要点 ★ |
|------|--------|
| 物质 | 状态变化、溶解、电路、磁铁、光与声 |
| 生命 | 植物结构、动物分类、生态系统、人体系统 |
| 地球 | 天气气候、地表、日月季节 |
| 工程 | 简单机械、设计—制作—改进流程 |
